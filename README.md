# Instagrao - Clone do Instagram

Este projeto é uma cópia da interface da página inicial do Instagram para desktop, contendo um sistema de carrossel de stories, e um feed de postagens.

# Demonstração
![Demontração do projeto](clone_instagrao/InstagraoDemonstracao.gif)

## Tecnologias utilizadas

- HTML5;
- CSS3;
- JavaScript;
- React;

## Funcionalidades

- [X] Carrossel de stories com fluxo horizontal;
- [X] Animações dinâmicas feitas em CSS;
- [X] Feed de postagens;

## Como rodar o projeto
### ⚙️ Pré-requisitos

Antes de rodar a aplicação, você precisa ter instalado:
- [Node](https://nodejs.org/)
- npm(Já incluso no Node)

### 💿 Rodando o projeto
No Terminal do git, faça:

#### 1. Clone o repositório
`git clone https://github.com/pedypowgui/CloneInstagrao.git`

#### 2. Acesse a pasta
`cd CloneInstagrao`

#### 3. Instale as dependências
`npm install`

#### 4. Rode o projeto
`npm run dev`

#### 5. Acesse no navegador
`http://localhost:5173 (Link padrão do Vite)`
